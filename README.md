# MADRAS BOXING CLUB

A premium cinematic landing page for Madras Boxing Club — an elite modern boxing academy based in Chennai, Tamil Nadu.

## Design Philosophy

- Dark luxury sports aesthetic
- Cinematic atmosphere
- Aggressive typography
- High contrast
- Deep red accent lighting
- Premium sports branding
- Editorial composition

## Tech Stack

- Next.js 14
- Tailwind CSS
- Framer Motion
- Lenis (smooth scroll)
- TypeScript

## Color Palette

- Matte Black: #080808
- Carbon Gray: #141414
- Blood Red: #D90429
- Fight White: #F5F5F5
- Steel Gray: #8A8A8A
- Gold Accent: #C9A227

## Typography

- Headings: Bebas Neue / Oswald
- Body: Inter

## Features

- Full-screen cinematic hero with parallax
- Layered typography behind boxer silhouette
- Smooth scroll with Lenis
- Scroll-triggered animations
- Particle background effects
- Glassmorphism contact form
- Animated statistics counters
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
