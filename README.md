# KTS BOXING CLUB

Premium cinematic combat sports website built for KTS Boxing Club, Chennai.

A modern high-impact boxing academy experience focused on:
- elite branding
- athlete culture
- lead generation
- immersive visuals
- luxury combat-sports presentation

---

# About The Project

This project is designed as a premium digital identity for KTS Boxing Club.

Instead of a generic gym website, the experience focuses on:
- cinematic storytelling
- emotional sports branding
- premium motion design
- high-conversion CTA sections
- modern combat-athlete aesthetics

The website is structured as a scalable foundation that can later evolve into:
- membership systems
- student dashboards
- payment integration
- admin management
- booking systems

---

# Core Features

- Cinematic fullscreen Hero section
- Premium animated typography
- 3D rotating fighter carousel
- Luxury black + gold combat aesthetic
- Responsive modern UI
- Scroll-triggered animations
- Smooth scrolling experience
- WhatsApp lead generation CTA
- Programs showcase
- Testimonials section
- Academy timing section
- Modern glassmorphism effects

- Performance-optimized animations
